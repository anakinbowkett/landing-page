Rotation

Turning a shape around a fixed point by an exact angle

What This Is

Rotation means turning a shape around a fixed point called the centre of rotation. You spin every point of the shape through a given angle — either clockwise or anticlockwise — keeping every point the same distance from the centre. The shape stays exactly the same size and the same way round; it just ends up in a new position. On the coordinate grid you use the rule (x, y) → (y, −x) for 90° clockwise or the rule (x, y) → (−x, −y) for 180°.

Key Terms You Must Know

Centre of Rotation

The fixed point that the shape turns around. Every other point in the shape moves; this one point stays still. It is usually given as a coordinate, most often the origin (0, 0).

Angle of Rotation

How far the shape turns, measured in degrees. The most common angles are 90°, 180°, and 270°. A full turn is 360° and brings the shape back to where it started.

Direction

Which way the shape turns — clockwise (same direction as clock hands) or anticlockwise (the opposite direction). You must always state the direction unless the angle is 180°, which gives the same result either way.

Object and Image

The object is the original shape before the rotation. The image is the shape after the rotation. Label image vertices with a dash: A becomes A′ (called "A prime"), B becomes B′, and so on.

Invariant Point

A point that does not move during the transformation. When rotating, the only invariant point is the centre of rotation itself. Every other point moves to a new position.

The Method — Step by Step
Grade 3–5 topic. Rotation appears on both Foundation and Higher papers. You need to be able to draw a rotation and also describe one fully.
1

Mark the centre of rotation on the grid

Read the question to find the centre of rotation. It will be given as a coordinate, such as (0, 0), (1, 1), or (2, −1). Put your pencil on that exact point and mark it clearly with a dot or a small cross so it stays visible throughout your working.

The centre of rotation is the one point in the entire diagram that does not move. Every other point in the shape will travel in a circular arc around it. Keeping track of this fixed point prevents your whole image from ending up in the wrong place.

On the worked example, the centre of rotation is the origin (0, 0). Mark this with a bold dot before touching the triangle.
2

Read and write down the angle and direction

Read the question and write the angle (90°, 180°, or 270°) and the direction (clockwise or anticlockwise) at the top of your working. These two pieces of information control where every single point ends up — getting either one wrong flips the entire image to the wrong side of the grid.

Notice that a 90° clockwise rotation and a 270° anticlockwise rotation both land in the same place. The question will give you one phrasing — use whichever words it gives. Write both pieces of information as a heading so you can check them before you draw.

The worked example rotation is 90° clockwise about (0, 0). Write this heading before drawing anything.
3

Apply the coordinate rule to every vertex

Choose the correct rule for your rotation. For 90° clockwise about the origin: (x, y) → (y, −x). For 180° about the origin: (x, y) → (−x, −y). For 90° anticlockwise about the origin: (x, y) → (−y, x). Write the rule at the top of a small table so every vertex gets treated the same way.

Apply the rule to each vertex one at a time. Take vertex A at (1, 2). The rule says: new x = old y = 2, new y = negative old x = −1. So A maps to A′ at (2, −1). Work through every vertex in the table before you plot anything. This keeps the arithmetic separate from the drawing.

Triangle with A(1,2), B(3,2), C(1,4). Applying (x,y)→(y,−x): A′(2,−1), B′(2,−3), C′(4,−1).
4

Plot each image vertex and join them up

Plot each new coordinate from your table onto the grid. Work across the columns of your table one vertex at a time — plot A′, then B′, then C′. Mark each image point with a small cross and write its label beside it so you can connect them correctly in the next move.

Connect A′, B′, and C′ with straight lines in the same order you connected A, B, and C. The image triangle must be the same shape and size as the object. If yours looks stretched or squashed, check your coordinate rule — a common slip is forgetting the negative sign in −x.

Plot A′(2,−1), B′(2,−3), C′(4,−1) and join them. The image triangle sits in the fourth quadrant, rotated 90° clockwise from the original.
5

Label the image and write the full description

Label each image vertex with a prime: A′, B′, C′. This tells the examiner which vertex maps to which, and it earns the labelling marks. Never label the image with the same letters as the object — that would imply the shape has not moved.

If the question asks you to describe a rotation, you must give three things: the transformation name ("rotation"), the angle and direction (e.g. "90° clockwise"), and the centre of rotation as a coordinate (e.g. "(0, 0)"). Leave out any one of these three and the examiner will not award full marks.

State: "Rotation, 90° clockwise, about (0, 0)." All three facts present — full description marks secured.
6

In the event of a centre that is NOT the origin — translate first

If the centre of rotation is a point other than (0, 0) — say (2, 1) — you cannot use the standard coordinate rule directly. Shift every vertex so that the given centre moves to the origin: subtract the centre's x-coordinate from every x, and subtract the centre's y-coordinate from every y. Now apply the normal rotation rule. Then shift back by adding the centre coordinates again. This three-step process handles any centre without needing a different formula.

Centre (2,1), vertex A(3,3): subtract → (1,2); rotate 90° CW → (2,−1); add centre back → A′(4,0).

Clockwise and anticlockwise give images on opposite sides of the origin!

1 / 3 marks
Rotating anticlockwise instead of clockwise places the entire image in the wrong quadrant. The 90° clockwise rule is (x, y) → (y, −x) — the new y-value is the negative of the old x. The anticlockwise rule gives a positive x, producing a mirror image across the y-axis. Direction is not interchangeable. Always check which way the clock hands turn, then choose the matching rule before writing a single coordinate.
