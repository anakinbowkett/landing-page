Vector Addition and Subtraction

Add and subtract column vectors, and understand what the result represents as a journey on a grid

What This Is

A vector describes a movement — how far right or left, and how far up or down. When you add two vectors you are chaining two journeys together to find the single journey that has the same effect. When you subtract, you reverse one of the journeys before chaining. The arithmetic is simple column-by-column addition or subtraction — the challenge is understanding what the column vector means and how the diagram matches the numbers.

Key Terms You Must Know

Vector

A quantity that has both a size and a direction. In GCSE maths, vectors describe movements on a grid. A column vector has two numbers: the top number is how far right (positive) or left (negative), and the bottom number is how far up (positive) or down (negative).

Column Vector

The way a vector is written in GCSE: one number stacked above another inside brackets. The top number always gives the horizontal movement (x), and the bottom number always gives the vertical movement (y). For example, a vector of 3 right and 1 up is written as a column with 3 on top and 1 below.

Resultant Vector

The single vector that has the same overall effect as two or more vectors combined. If you travel along vector a then along vector b, the resultant is the direct journey from your start to your end — it represents the shortcut.

Negative Vector

The vector that goes in exactly the opposite direction. To find the negative of a vector, flip both signs: the top and bottom numbers both change sign. The negative of a vector with 1 on top and −2 below is −1 on top and 2 below. This is what you use when subtracting vectors.

Scalar Multiple

When you multiply a vector by a number (a scalar), you multiply both the top and bottom numbers by that number. The direction stays the same but the length scales up. Multiplying a vector by 2 doubles every number in the column.

The Method — Step by Step
Grade 4–6 topic. On the foundation paper, you will be asked to add or subtract two column vectors, find a resultant, or calculate a scalar multiple. Every question uses the same column-by-column rule below.
1

Read a column vector — the top number is sideways, the bottom number is up or down

Look at the top number of the column vector. A positive top number means move right. A negative top number means move left. The size of the number is how many squares you travel horizontally.

Look at the bottom number. A positive bottom number means move up. A negative bottom number means move down. A vector with top 3 and bottom −2 means: go right 3, then go down 2.

Example: vector a = (3 over 1) means go right 3 and up 1. Vector b = (1 over −2) means go right 1 and down 2. Now add them together.
2

Add two column vectors — add the top numbers together, then the bottom numbers separately

Write the two column vectors next to each other with a + sign between them. Add the two top numbers together to get the top of the answer. These are the horizontal movements: you are combining all the left-right travel into one total.

Add the two bottom numbers together to get the bottom of the answer. These are the vertical movements: take care with negatives — adding a negative number means subtracting. 1 + (−2) = −1, which means the overall vertical movement is 1 down.

Example: a + b = (3 over 1) + (1 over −2) = (4 over −1). Top: 3+1=4. Bottom: 1+(−2)=−1. Result is right 4, down 1.
3

Check your answer with a diagram — draw the vectors tip-to-tail and the resultant goes straight from start to finish

Draw vector a first: start at a point and draw an arrow going right 3, up 1. The tip of arrow a is where arrow b starts — place the tail of b at the head of a. Draw b from there: right 1, down 2. The tip of b is where the combined journey ends.

Draw the resultant vector as a single arrow from the very start to the very end. This resultant is the column vector you calculated by adding: (4 over −1). It cuts directly across rather than going in two legs.

Example: a + b = (4 over −1). Draw a then b tip-to-tail. The resultant arrow from start to end goes right 4, down 1 — which is exactly (4 over −1).
4

Subtract vectors — flip the signs of the second vector, then add as normal

To find a − b, first find −b by flipping both signs in vector b. If b has top 1 and bottom −2, then −b has top −1 and bottom 2. Flipping the signs reverses the direction of travel — instead of right 1 down 2, you now go left 1 up 2.

Now add a and −b using the same rule as Step 2: add top to top, bottom to bottom. This gives the result of a − b without needing to think of subtraction separately. Top: 3 + (−1) = 2. Bottom: 1 + 2 = 3.

Example: a − b = (3 over 1) − (1 over −2) = (3 over 1) + (−1 over 2) = (2 over 3). Right 2, up 3.
5

Multiply a vector by a number — multiply every number in the column by that number

To find 2a, multiply both numbers inside the column vector by 2. If a = (3 over 1), then 2a = (6 over 2). The direction stays exactly the same — the arrow simply becomes twice as long. This is called a scalar multiple.

Example: 2a = 2 × (3 over 1) = (6 over 2). Both numbers multiplied by 2. Arrow twice as long, same direction.
6

In the event of a combined expression — apply scalar multiples before adding or subtracting

Some questions give an expression like 2a + 3b or a − 2b. Deal with scalar multiples first: work out what 2a is, work out what 3b is, then add the two resulting column vectors together. Never add the multipliers to the vectors first.

After applying scalar multiples, add column by column as normal. Watch for a zero result in one row — a zero top means no horizontal movement, a zero bottom means no vertical movement. The vector still exists: it just points purely left/right or purely up/down.

Example: 2a + b = (6 over 2) + (1 over −2) = (7 over 0). Purely horizontal — the vertical movements cancel.
