Finding the nth Term

Write a formula that gives any term of a linear sequence

What This Is

The nth term is a formula that lets you calculate any term in a sequence without listing every term before it. You substitute a position number — 1, 2, 3, and so on — into the formula and it produces the term at that position. For a linear sequence (one where the difference between consecutive terms is always the same), the nth term always takes the form dn + c, where d is the common difference and c is a constant you calculate. Your job is to find d and c from the sequence you are given.

Key Terms You Must Know

Linear Sequence

A sequence where the same number is added (or subtracted) each time. The differences between consecutive terms are all equal. Examples: 3, 7, 11, 15 (add 4 each time) and 20, 17, 14, 11 (subtract 3 each time).

Common Difference (d)

The fixed amount added from one term to the next. Found by subtracting any term from the one after it. This becomes the coefficient of n in the nth term formula.

nth Term

A formula written in terms of n that generates the term at position n. For a linear sequence it takes the form dn + c. Substituting n = 1 gives the first term, n = 2 gives the second, and so on.

Position Number (n)

The counting number that labels each term's place in the sequence. The first term is at position n = 1, the second at n = 2, and so on. n is always a positive integer.

Zero Term

The term that would come before the first term if the sequence were extended backwards one step. Subtracting the common difference from the first term gives it. It equals c in the formula dn + c.

The Method — Step by Step
Grade 4–5 topic. Finding the nth term of a linear sequence appears on nearly every Foundation paper, often worth 2–3 marks, and frequently reappears inside harder questions about sequences and graphs.
1

Write out the sequence with position numbers above each term

Write the terms of the sequence in a row and label each position: n = 1 above the first term, n = 2 above the second, and so on. The position label is not decoration — it is the value you will substitute into your formula when you check it at the end.

e.g. Sequence: 5, 8, 11, 14, 17 — label n = 1 under 5, n = 2 under 8, and so on.
2

Find the common difference — this is d, the coefficient of n

Subtract the first term from the second term. That result is the common difference. Write it in a gap between each pair of consecutive terms — all the gaps must be identical, which confirms the sequence is linear and your subtraction is correct.

Write d = 3 and note that your nth term starts with 3n. The coefficient of n is always the common difference — because each time n increases by 1, the term increases by exactly d.

e.g. 8 − 5 = 3, so d = 3 and the formula starts 3n.
3

Find c — subtract d from the first term to get the zero term

The zero term is the value that would come at position n = 0 — one step before the first term. Subtract the common difference from the first term to find it. This value becomes c, the constant in your formula.

Write c = zero term. The reason this works is that when n = 1 in the formula dn + c, you get d + c, and that must equal the first term. So c = first term − d. Both routes give the same result — always check that they agree.

e.g. c = 5 − 3 = 2, so the constant is +2.
4

Assemble the formula as dn + c and write it clearly

Place d in front of n, then write + c after it. If c is negative, write − |c| — never write + −2. The formula is complete when it contains no extra labels, no equals sign on the right, just the expression in n.

e.g. nth term = 3n + 2
5

Check — substitute n = 1 and n = 2 into your formula and verify the results match the original sequence

Substitute n = 1 into your formula. The result must equal the first term of the sequence. If it does not, your value of c is wrong — go back to Step 3.

Substitute n = 2. The result must equal the second term. Two successful checks confirm the formula is correct. The mark scheme requires the formula, not the check — but the check takes ten seconds and eliminates any careless error before you move on.

e.g. n = 1 → 3 + 2 = 5 ✓  |  n = 2 → 6 + 2 = 8 ✓
6

In the event of being asked to use the nth term — substitute, rearrange, or solve

If asked for a specific term — such as the 50th — substitute n = 50 directly into the formula and evaluate. There is no need to list 50 terms; the formula gives the answer in one step.

If asked whether a given value — say 50 — is in the sequence, set the formula equal to that value and solve for n. If n is a positive whole number, the value is in the sequence. If n is a fraction or negative, it is not.

e.g. Is 50 in the sequence? 3n + 2 = 50 → n = 16. Yes — it is the 16th term.

Describing the term-to-term rule — "add 3 each time" — answers a different question entirely. The nth term must be an algebraic expression containing n, not a description of how the sequence grows. The common difference is still useful — it becomes the coefficient of n — but the formula must allow you to calculate any term directly from its position number. A sequence rule tells you how to get from one term to the next; the nth term formula lets you jump straight to any position without listing every term before it. Write 3n + 2, never "add 3".
