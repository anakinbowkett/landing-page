Sequences: Position-to-Term Rules

Find the nth term of a linear sequence — then use it to generate or test any term.

What This Is

A position-to-term rule — also called the nth term — is a formula that lets you calculate any term in a sequence directly from its position number. Instead of counting along the sequence one step at a time, you substitute the position number n into the formula and work it out. For a linear sequence (one that goes up or down by the same amount each time), the nth term always takes the form an + b, where a is the common difference and b is an adjustment. Finding that formula and using it are the two skills this guide teaches.

Key Terms You Must Know

Term

A number in the sequence. The 1st term is the first number, the 2nd term is the second, and so on.

Position (n)

The place number of a term in the sequence. The 1st term has position n = 1, the 2nd has n = 2, and so on.

Common difference (d)

The fixed amount added (or subtracted) from one term to get to the next in a linear sequence. In 7, 11, 15, 19, … the common difference is +4.

nth term

A formula in terms of n that gives the value of any term when you substitute the position number. Written as expressions like 4n + 3 or 2n − 5.

Linear sequence

A sequence with a constant common difference — the terms form a straight line when plotted. The nth term of a linear sequence always has the form an + b.

The Method — Step by Step
Grade 4–5 topic. Finding and using the nth term is one of the most reliably tested topics on AQA Foundation. Learn this method and you will pick up full marks every time it appears.
1

Find the common difference by subtracting consecutive terms

Write out the sequence in a row and subtract each term from the one that follows it. Do this for every consecutive pair you are given — all the gaps must be equal for the sequence to be linear.

The common difference is the coefficient of n in the nth term. Because each term increases by 4, the nth term begins with 4n. Write this down before moving on.

e.g. Sequence 7, 11, 15, 19, … → differences are all +4 → nth term starts 4n.
2

Write out the multiples of the common difference under the sequence

Write the sequence of multiples of the common difference starting at position 1: 4×1, 4×2, 4×3, 4×4 — giving 4, 8, 12, 16. Write these directly below the original sequence so the positions line up.

Compare the original sequence to the 4n sequence at every position. The original is consistently higher (or lower) by the same fixed amount — that gap is the adjustment b you need to add.

e.g. 7 − 4 = 3, 11 − 8 = 3, 15 − 12 = 3 → the adjustment is always +3.
3

Write the nth term as: (common difference) × n + (adjustment)

Combine the two pieces: the coefficient of n (the common difference) and the constant term b (the gap you just found). Write the nth term formula in the form an + b.

Verify by substituting n = 1: 4(1) + 3 = 7 ✓. Check n = 2: 4(2) + 3 = 11 ✓. Both match the sequence, so the formula is correct. Always check at least two positions before writing your final answer.

e.g. nth term = 4n + 3. Check: n=1 → 7, n=2 → 11. Both match — the formula is correct.
4

Use the nth term to find a specific term or to test whether a number is in the sequence

To find a specific term — say the 20th — substitute n = 20 into the formula: 4(20) + 3 = 83. You do not need to count along the sequence. This is exactly what the nth term is for.

To test whether a given number is in the sequence — say 47 — set the formula equal to that number and solve for n. If n is a positive whole number, the number is in the sequence at that position. If n is a fraction or negative, it is not in the sequence.

e.g. Is 47 in the sequence? 4n + 3 = 47 → 4n = 44 → n = 11. Whole number, so yes — 47 is the 11th term.
5

In the event of a decreasing sequence — the common difference is negative

If the sequence decreases — for example 20, 17, 14, 11, … — the common difference is negative. Subtract consecutive terms in order: 17 − 20 = −3. The common difference is −3, so the nth term starts −3n.

Apply the same method: the −3n sequence gives −3, −6, −9, −12. Compare to the original: 20 − (−3) = 23, 17 − (−6) = 23. The constant b is +23. The nth term is −3n + 23.

e.g. 20, 17, 14, 11 → d = −3 → −3n sequence is −3, −6, −9, −12 → gap = +23 → nth term = −3n + 23.

d = 4 → nth term = 4n + ?

STEP 2 — 4n sequence

Write the multiples of 4 under the sequence: 4, 8, 12, 16. These are the values 4n produces at n=1, 2, 3, 4. Compare each pair to find the fixed gap between them.

4n: 4, 8, 12, 16
Step 3 Find the constant b AQA
STEP 3 — Find b

Subtract the 4n value from the original term at each position: 7−4=3, 11−8=3. The gap is a constant +3 at every position, so b = 3. The constant must be the same everywhere — if it varies, recheck the common difference.

7 − 4 = 3 → b = 3
✓M1

STEP 4 — Write the formula

Combine the two parts: the coefficient 4 goes in front of n, and the constant 3 is added. Write nth term = 4n + 3. This is the position-to-term rule for the entire sequence.

nth term = 4n + 3

STEP 5 — Verify and answer
