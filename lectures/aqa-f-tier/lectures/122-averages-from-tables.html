Averages from Tables

Finding the mean, median and mode from a frequency table

What This Is

A frequency table is a compact way to record data — instead of listing every single value, it groups each value with a count of how often it appears. To find the mean from a table you add a third column, multiply each value by its frequency, total everything up, then divide. The median and mode can be read directly from the table once you know which row to look at. This topic appears on almost every AQA paper in some form.

Key Terms You Must Know

Frequency (f)

How many times a value appears in the data. The frequency column tells you how many items have each value.

f × x (frequency × value)

The product of a value and its frequency. Adding all these products gives you the total of all the data combined.

Σf (sum of frequencies)

The total number of data items — found by adding up the entire frequency column. This is the number you divide by to find the mean.

Mean

The sum of all values divided by how many there are. From a table: mean = Σ(fx) ÷ Σf.

Mode

The value that appears most often — the one with the highest frequency in the table.

Median

The middle value when all items are in order. From a table, find the position of the middle item using (Σf + 1) ÷ 2, then count through the frequencies to find which row that falls in.

The Method — Step by Step
Grade 3–4 topic. You will always be given the frequency table in the question. You add the fx column yourself in your working.
1

Read the table — identify the value column (x) and the frequency column (f)

Look at the table in the question. Find the column that lists the data values — this is x. Find the column that shows how many times each value occurs — this is f. These two columns are always given to you.

Write a heading for a new third column next to the table — label it "fx" or "f × x". This column is where you will record each value multiplied by its frequency. The table does not give you this column — you create it yourself.

Example: The table gives values 1, 2, 3, 4 with frequencies 3, 5, 4, 6. Add a blank "f × x" column next to it.
2

Fill in the fx column — multiply each value by its frequency

Go row by row. For each row, take the value in the x column and multiply it by the frequency in the f column. Write the result in the fx column for that row. Think of it as: "this value appeared this many times — so it contributes this much to the total."

Repeat this for every row until the fx column is fully filled. Each row gives you one multiplication: value × its frequency. After all four rows are done, the fx column should contain four numbers — one per row.

Example: Row 1: 1×3=3. Row 2: 2×5=10. Row 3: 3×4=12. Row 4: 4×6=24. Your fx column now reads 3, 10, 12, 24. Now total both columns.
3

Total the f column and total the fx column separately

Add up all the numbers in the f column. Write the total at the bottom of that column. This total is Σf — the total number of data items. In the example: 3 + 5 + 4 + 6 = 18.

Now add up all the numbers in the fx column. Write that total at the bottom of the fx column. This is Σ(fx) — the grand total of all values combined. In the example: 3 + 10 + 12 + 24 = 49.

Example: Σf = 3+5+4+6 = 18. Σfx = 3+10+12+24 = 49. Now divide them.
4

Divide Σ(fx) by Σf to find the mean

Write the formula: mean = Σ(fx) ÷ Σf. Substitute your two totals: divide the total of the fx column by the total of the f column. This gives you the mean — the average value of all the data.

Check whether the question asks for a specific degree of accuracy. If not, give the exact value or round to 2 decimal places. Write "mean = " before your answer — the examiner expects to see the label.

Example: Mean = 49 ÷ 18 = 2.72 (rounded to 2 d.p.). Write the full calculation on your exam paper.
5

In the event of a median or mode question — read these directly from the table

Find the mode by scanning the frequency column for the highest number. The value in the x column on the same row is the mode — the most common value. No calculation needed.

Find the median position using (Σf + 1) ÷ 2. Then count through the frequency column from the top until you reach that position — the value in that row is the median. You are locating the middle item in an ordered list, without writing out every single item.

Example: Σf = 18, so median position = 9.5th item. Items 1–3 have value 1, items 4–8 have value 2, items 9–12 have value 3. The 9.5th item falls in the x = 3 row, so median = 3.
