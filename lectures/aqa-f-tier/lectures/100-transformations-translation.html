Translation

Moving a shape exactly — same size, same angle, new position

What This Is

A translation slides a shape to a new position on a grid. Every point on the shape moves the same distance in the same direction — nothing rotates, nothing stretches, nothing shrinks. You describe a translation using a column vector, which tells you how far to move right or left and how far to move up or down. The shape you get after the translation is called the image.

Key Terms You Must Know

Translation

A transformation that slides a shape to a new position. The shape does not turn, flip or change size — it only moves.

Column Vector

The instruction that describes a translation. Written as two numbers stacked vertically: the top number is the horizontal move (right is positive, left is negative) and the bottom number is the vertical move (up is positive, down is negative).

Image

The shape in its new position after the translation. It is always labelled with a dash or a prime symbol — for example, A′ is the image of point A.

Object

The original shape before it is moved. The object and the image are exactly the same size and shape — only their position on the grid changes.

Vertex (plural: Vertices)

A corner of a shape. When you translate a shape, you move every single vertex by the same column vector, then join them up to form the image.

The Method — Step by Step
Grade 1–3 topic. This is one of the most straightforward transformations. Get the direction signs right and move every corner — those are the two things that cost marks.
1

Read the column vector — top number is sideways, bottom number is up or down

Look at the column vector written in the question. The top number tells you how far to move horizontally. A positive top number means move right; a negative top number means move left.

Look at the bottom number. A positive bottom number means move up; a negative bottom number means move down. In the vector shown, −2 means move down 2 squares.

2

Pick one corner of the shape and move it using the top number first

Choose any corner (vertex) of the original shape and mark it clearly. Count squares left or right using the top number of the vector. Mark a dot at the new position.

3

From the new dot, apply the bottom number — move up or down

Starting from the dot you placed in Step 2, count squares up or down using the bottom number of the vector. This gives you the final position of that corner on the image. Mark a dot there and label it A′.

4

Repeat for every other corner, then join the dots to draw the image

Apply the same column vector to every remaining corner of the object. Each corner moves the same amount — right 3 and down 2 in our example. Mark each translated corner with a dot and label it.

Join the translated dots in the same order as the original corners. The image is congruent to the object — it is identical in size and shape. Label the image clearly.

5

In the event of describing a translation — write the column vector, not words

Some questions show the object and image already drawn and ask you to describe the translation. Choose any corner on the object and find the matching corner on the image. Count squares right or left (this is your top number) then count squares up or down (this is your bottom number).

Write your answer as a column vector — not as "3 right and 2 down." The exam mark scheme always requires the column vector form. Writing words alone will not earn the accuracy mark.
